## 阶段门禁

> Bootstrap 初始化、迭代关闭确认和收尾归档不是迭代内的阶段，而是独立的非角色机制。触发时机和执行规则见 `docs/baseline/mechanisms.md`。
> 本区只放标准阶段标题：`PRD 阶段`、`设计阶段`、`实现阶段`、`部署就绪检查`。讨论记录、未来方向、停放性事项不得作为本区三级标题；应放入 `Review 记录`、`Change Notes` 或独立 ad-hoc。
> 阶段门禁状态只允许：`待Review / Review中 / 修改中 / 已定稿 / 阻塞 / 已跳过`。部署就绪检查使用独立状态：`待检查 / 检查中 / 通过 / 阻塞 / 已跳过`。门禁表最新轮次置于末行。
