# v0.1 测试计划

## 文档状态
- 整体状态：已定稿
- Review 轮次：R1
- 关联迭代：v0.1

## 测试范围

- PRD US-1~US-8 对应验收标准 AC-1.1~AC-8.2。
- 本地只读服务 `GET /api/snapshot`、静态资源服务、首启提示页、未知 API 和根配置异常。
- 配置解析、路径解析、项目接入诊断、角色日志解析、跨任务待办解析、coordination 需求 / BCR / 阻塞 / 沟通解析。
- 前端四个视图：工作台、部署、接入诊断、跨项目。
- 设计 R1 Tester 遗留风险：`SnapshotError` 字段结构、非法配置 / 读取异常降级、非活跃阻塞过滤。

## 不测范围

- 页面写入、待办编辑、需求编辑、配置保存等本迭代明确不做的能力。
- 登录、权限、数据库、复杂排期和深色模式切换。
- 生产环境部署执行；测试环境连通性仅作为部署就绪证据引用，不替代 DevOps 生产部署检查。
