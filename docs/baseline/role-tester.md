# Tester（测试工程师）角色 — 已废弃（retired）

本角色于 2026-06-25（BCR-006）废弃。质量保障改为：**Developer 自测**（接口/自动化测试 + 证据，见 `role-developer.md` §自测）+ **Owner 验收**（手动点击验收，迭代关闭门禁）；「验收标准 / 边界 / 回归」独立复核由 **Architect 或 DevOps** 承担（PM 产出验收标准时不自审）。

- 不再是可切换的工作流角色；入口触发表已移除「测试 / Tester」（"测试/QA"关键词指向 Developer 自测）。
- 历史测试产物（`vX.Y-test-plan.md`、`vX.Y-test-report.md`、`roles/tester.md` 等）保留存档、不追溯改写。
- 本文件为墓碑，仅供历史引用定位；`measure-context.sh` 不计入角色集。
